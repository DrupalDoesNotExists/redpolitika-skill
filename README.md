# Это — ИИ для Редполитики

В этом репозитории — скилл для ИИ-агентов. Он научит вашего агента писать правила для Редполитики. Подойдёт тем, кто не разбирается в конфигах, ИТ:
1. скопируйте ссылку и попросите агента установить скилл;
2. объясните агенту требования или скиньте свою редполитику в пдф;
3. попросите написать правила для Редполитики;
4. подключите правила к сервису.

**Подойдёт пользователям Коммьюнити и Энтерпрайза**. Скилл описывает и формат правил из ядра, и встроенные плагины из EE-образа.
